# Contribuer à repod

Merci de votre intérêt pour repod !

## Accord de contribution (CLA)

repod utilise un modèle de **double licence** (AGPL-3.0 + Licence Commerciale).
Pour maintenir ce modèle, toute contribution externe nécessite l'acceptation
du Contributor License Agreement décrit dans [CLA.md](CLA.md).

**Comment accepter le CLA :**

Ajoutez cette ligne dans votre pull request ou message de commit :

```
J'ai lu et j'accepte le CLA de repod v1.0 (CLA.md)
```

Les contributions sans cette mention ne pourront pas être fusionnées.

## Processus de contribution

1. Forkez le dépôt
2. Créez une branche : `git checkout -b fix/mon-correctif`
3. Committez vos changements avec l'acceptation CLA dans le message
4. Soumettez une pull request en décrivant le problème résolu

## Signalement de vulnérabilités
